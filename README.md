览山 - "方寸屏间，万里河山"
项目简介
"览山"是一个融合中华文化与现代科技的创新网页设计项目，以"方寸屏间，万里河山"为主题，通过数字化的方式展现中国大地的壮美风光和深厚文化底蕴。项目采用纯HTML、CSS、JavaScript技术栈打造，提供沉浸式的视觉体验和交互式地图探索功能。
主题选择
"方寸屏间，万里河山" 晨光漫过黄山之巅，劲松破石耸立；云海翻上岱宗山顶，奇峰凌绝天下。黄河劈开黄土秦关，浊浪拍打着千百年的岩壁；长江绕出古蜀三峡，波涛晕染出水墨色的江南。北国的霜雪，南海的热风，珠峰日照金山的奇观下，祖国画卷徐徐展开……本项目以创意为墨，代码为笔，用屏幕装下祖国的恢宏与瑰丽，让用户感受到咫尺天涯的视觉冲击。

技术栈：
前端框架：原生HTML5 + CSS3 + JavaScript ES6+
数据可视化：ECharts 5.x
地图数据：China.js/China.json (GeoJSON格式)
动画效果：HTML5 Canvas + CSS3动画
响应式设计：支持1920×1440×720分辨率
第三方库及用途：
1. ECharts (v5.x)
用途：中国地图可视化和数据展示
应用场景：
交互式中国地图展示
森林覆盖率数据可视化
水资源数据可视化
6大区域（东北、华北、华东、华南、华中、西北）信息展示
版本：5.x
优势：专业的图表库，提供丰富的地图交互功能和自定义样式能力
2. China.js / China.json
用途：中国地图GeoJSON数据文件
应用场景：
提供中国34个省市自治区的地理坐标数据
支持ECharts地图图表渲染
省份边界绘制和区域划分
数据覆盖：全国34个省、自治区、直辖市、特别行政区
用途说明：标准的地图数据文件，为地图可视化提供地理信息基础
核心功能：
1. 山河卷全屏展示
全屏沉浸式浏览体验；
6大区域切换：关东霜原、燕赵之地、楚泽云川、江南锦域、南粤山海、云岭高原；
优雅的过渡动画和视觉特效；
鼠标/触摸交互支持；
2. 数字中国行 - 交互式地图
虚拟中国模式：
6大区域可视化展示；
点击省份查看区域详情；
区域文化介绍和风光展示；
智慧守护模式：
森林覆盖率数据可视化；
水资源数据可视化；
实时数据图表切换；
地图缩放、平移、交互功能；
3. 粒子动态背景
Canvas实现的科技感粒子系统；
鼠标交互效果（粒子跟随、连接线动画）；
蓝青色系渐变，营造科技氛围；
性能优化，60FPS流畅动画；
4. 响应式设计
自适应多种屏幕尺寸；
1920×1440×720三种主要分辨率适配；
移动端触摸手势支持；
跨浏览器兼容性保证；

直接在浏览器中打开 index.html，推荐使用Chrome浏览器（支持Chrome 129+），确保网络连接正常（加载地图数据和第三方库）
浏览器兼容性：Chrome 90+ ，Firefox 88+ ，Safari 14+ ，Edge 90+
